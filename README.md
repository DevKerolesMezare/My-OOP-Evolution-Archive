# My OOP Evolution Archive

## Important Note About Project Structure

Please note that the folder structure shown on GitHub may appear different from the structure inside Visual Studio.

This project was organized using Visual Studio Filters (virtual folders such as Core, Screens, ATM Project, Student Management System, and more) to maintain clean architecture and better readability during development.

GitHub only displays the real physical folders and files stored on disk, and it does not recognize Visual Studio Filters.

For the complete organized structure and the intended development experience, it is highly recommended to download the project and open it using Visual Studio.

This will allow you to explore the full architecture exactly as designed.

It contains numerous applications, problem-solving implementations, system designs, banking systems, ATM systems, student management systems, utility libraries, and many other real practice projects built during my learning process.

This archive reflects my evolution in C++ development, software design thinking, and OOP architecture implementation using Visual Studio.

It is not just a single project, but a complete development journey and practical learning archive.
